Projekt: ARKIST Motion AI

Ziel:
Entwicklung eines AR-gestützten Rehabilitations- und Bewegungstherapiesystems.

Aktueller Stand:

- MediaPipe Pose Tracking integriert
- 33 Landmarken werden in Echtzeit erfasst
- Speicherung als Numpy-Sequenzen (.npy)
- 3D Visualisierung mit Matplotlib
- JSON-Übungsdatenbank aus Excel aufgebaut
- Feedback-Engine für erste Übungen implementiert
- Regelbasierte Analyse (Jefferson Curl Prototyp)

Verfügbare Daten:

- Übungen_Demoprogramm.xlsx
- Übungen_Gesamtübersicht.xlsx

Extrahierte Daten:

- demo_raw.json
- overview_raw.json
- exercises_demo_structured.json

Projektideen:

1. Face Mesh Integration
2. Body Mesh Integration
3. Körperproportionsanalyse
4. Haltungserkennung
5. Avatar-Steuerung
6. AR-Brillenintegration
7. Unsupervised Feedback
8. D2STA Integration

Langfristige Vision:

Patient -> Tracking -> Analyse -> Feedback -> Progression -> AR-Therapie

Betreuerwünsche:

- Face Mesh
- Body Mesh
- Körperformanalyse
- Haltungserkennung
- automatische Feedbackgenerierung
- Unsupervised Learning

Wichtige Erkenntnis:

Die Excel-Tabellen sind Ausgangspunkt, dürfen aber biomechanisch
verbessert, erweitert und wissenschaftlich optimiert werden.
