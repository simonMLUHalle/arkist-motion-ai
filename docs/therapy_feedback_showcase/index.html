ARKIST Therapie-Feedback Demo

Die Bewegung wird automatisch erkannt. Kritische Signale werden hier nicht als Endurteil gezeigt, sondern als Qualitaets-Gate fuer therapeutisches Feedback. Die Animation laeuft automatisch, danach folgt eine kurze Lesepause mit dem wichtigsten Korrekturhinweis.

Was schon funktioniert

  • automatische Bewegungserkennung aus Smartphone-Landmarks
  • programmiert auf Basis von Python, MediaPipe Pose, NumPy, Pillow und lokalen HTML-Reports
  • technisches, psychologisches und sicherheitsbezogenes Feedback
  • unsupervised Mustererkennung ueber Zeitfenster

Was diese Demo zeigt

  • zwei reale Aufnahmen mit automatischer Bewertung
  • Proxy-Mesh, Hals-/Kopfbezug und Bewegungsphasen
  • Warnungen als Qualitaets-Gate statt als pauschales Stopp-Signal

Unsupervised im Modell

Aktuell unsupervised

  • lokales Clustering zeitlicher Bewegungsfenster
  • Zustandsmodellierung statt Einzelbildbewertung
  • Mustererkennung ohne Uebungslabels
  • Anbindung an technisches und psychologisches Feedback

Zukuenftig unsupervised

  • personenbezogene Baselines statt fixer Grenzwerte
  • Anomalieerkennung ueber laengere Verlaufsdaten
  • adaptive Risiko- und Freigabelogik
  • personenspezifisches Feedback je Leistungsstand und Tagesform

Roadmap Unsupervised KI

Perception Layer

  • Pose und Silhouette robust erfassen
  • Trackingqualitaet absichern
  • Proxy-Mesh stabilisieren

Representation Layer

  • Symmetrie, Tiefe, Stabilitaet, Tempo berechnen
  • Kopf-, Hals- und Rumpfhaltung strukturieren
  • Merkmalsraeume aufbauen

Unsupervised Layer

  • Bewegungszustaende ohne Labels clustern
  • Auffaellige Muster ueber Zeit erkennen
  • Personenspezifische Bewegungstypen entdecken

Expert Layer

  • Technische Warnungen ableiten
  • Therapeutische Regeln versionieren
  • Risiken transparent machen

Feedback Layer

  • Technisches Haltungsfeedback formulieren
  • Psychologisch motivierendes Feedback anpassen
  • Sicherheits- und Belastungshinweise staffeln

Learning Layer

  • Personenbezogene Baselines lernen
  • Anomalien frueh erkennen
  • Feedback datenbasiert verbessern
